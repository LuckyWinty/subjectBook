### 缓存分类

##### 缓存位置

* Service Worker

* Memory Cache

* Disk Cache

* 网络请求

**按照获取资源时请求的优先级依次排列**

1. Memory Cache

2. Service Worker Cache

3. HTTP Cache
4. Push Cache

## HTTP 缓存机制

HTTP 缓存是我们日常开发中最为熟悉的一种缓存机制。它又分为**强缓存**和**协商缓存**。优先级较高的是强缓存，在命中强缓存失败的情况下，才会走协商缓存。
